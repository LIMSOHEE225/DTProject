# 📊 AI 월간 학습 분석 보고서 시스템

## 1. 개요
단순 점수 나열이 아닌, 학생이 반복적으로 틀리는 '취약 개념'을 AI가 데이터 마이닝하여 선생님께 통찰력을 제공하는 정기 보고서 기능입니다.

## 2. 데이터 수집 아키텍처
1. **정답로그 테이블 (`solved_logs`)**:
   - 학생ID, 문제유형(예: 연합연산, 분수), 정오답 여부, 소요 시간, 오답 패턴을 기록합니다.
2. **Batch 프로세스 (Spring Boot)**:
   - 매월 말일, 특정 학급 학생들의 `solved_logs`를 취합합니다.
3. **AI 통계 처리 (Gemini API)**:
   - "A 학생은 기초 연산은 100점이지만, 괄호가 포함된 응용 문제에서 정답률이 20%로 급감함"과 같은 분석 텍스트를 AI로 생성합니다.

## 3. 리포트 구성 요소
- **Concept Cloud**: 많이 틀리는 개념 키워드 시각화
- **Trend Chart**: 주차별 성취도 변화 그래프
- **Focus Recommendation**: "이번 달은 이 학생에게 '사칙연산의 우선순위' 개념 보충이 필요합니다"라는 AI 처방전

## 4. 발송 트리거
- 선생님 대시보드 내 [월간 리포트 확인] 알림 배지 활성화 및 이메일/알림톡 발송 기능 연동.
