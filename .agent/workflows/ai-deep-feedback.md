# 💬 AI 심층 피드백 및 응원 시스템

## 1. 개요
단순히 맞고 틀림을 넘어, 학생의 '생각의 흐름'을 AI가 파악하여 왜 틀렸는지를 명확히 짚어주고 학습 동기를 유발하는 개인 맞춤형 인터페이스입니다.

## 2. 분석 로직 (Prompt Engineering)
AI 모델(Gemini)에게 학생의 답안 패턴을 전달할 때 아래와 같은 상세 문맥을 포함합니다.
- **입력 데이터**: [문제 내용, 정답, 학생이 선택한 답, 이전 회차 유사 문제 정답률]
- **요구 사항**: "이 학생이 (5-2)와 같은 괄호 안의 연산을 먼저 해야 한다는 '연산 우선순위'를 이해하지 못한 것인지, 아니면 단순 계산 실수인지 판별하여 따뜻한 문체로 조언해줘."

## 3. UI/UX 구성
1. **분석 카드**: 채점 직후 화면 중앙에 AI 로봇 아이콘과 함께 등장.
2. **응원의 한마디**: "○○아, 기초 연산은 완벽해! 이제 괄호라는 보물상자를 먼저 여는 연습만 하면 수학 왕이 될 수 있어!"와 같은 감성적 피드백 제공.
3. **취약 개념 링크**: 분석 결과와 연관된 보충 학습 페이지나 짧은 개념 영상을 즉시 추천.

## 4. 데이터 활용
- 생성된 피드백 텍스트는 DB에 저장되어, '월간 리포트' 생성 시 핵심 소스로 다시 활용됩니다.
